# ⚡ Solar Inverter System

A simple PHP project that helps manage power sources automatically for an inverter system.

---

## 👋 What this project does

This system automatically chooses where power should come from so things don’t get complicated or overloaded.

It can switch between:
- Direct Current (DC)
- Inverter
- Generator

You can also take control manually if you want.

---

## 🧠 Key Features

- 🔄 Auto-switching between power sources  
- 🎛 Manual override (you choose the power source)  
- ⚖️ Load balancing to prevent inverter overload  
- 🧠 Simple smart decision logic  

---

## 🛠 Built With

- PHP (just one file: `index.php`)

## 🚀 How to Run It

1. Clone the project:
```bash
git clone https://github.com/jamesallison965/Solar-Inverter-Systems.git
```

2. Move it into your server folder:
- XAMPP → `htdocs`
- WAMP → `www`

3. Start your server (Apache)

4. Open in browser:
```plaintext
http://localhost/Solar-Inverter-Systems
```
